# 📉 Depreciation Calculator

## 📖 Overview

This Microsoft Excel project calculates asset depreciation using multiple depreciation methods. It helps businesses estimate the reduction in asset value over time and supports financial planning and accounting decisions.

---

## 🎯 Objectives

- Calculate asset depreciation
- Compare different depreciation methods
- Analyze yearly depreciation expenses
- Support financial reporting

---

## ✨ Features

- Straight-Line Method (SLM)
- Declining Balance Method (DBM)
- Double Declining Balance (DDB)
- Sum-of-the-Years'-Digits (SYD)
- Automatic Year-wise Depreciation Schedule
- Dynamic Excel Formulas
- Easy-to-Use Input Section
