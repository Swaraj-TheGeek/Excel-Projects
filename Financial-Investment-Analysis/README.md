# 📈 Financial Investment Analysis

## 📖 Overview

This project demonstrates financial investment analysis using Microsoft Excel. It includes financial statement analysis, investment appraisal techniques, and data visualization to support business decision-making.

---

## 🎯 Objectives

- Analyze financial performance
- Evaluate investment opportunities
- Calculate key financial metrics
- Support investment decisions using Excel

---

## ✨ Features

- Financial Statement Analysis
- Ratio Analysis
- Trend Analysis
- Net Present Value (NPV)
- Internal Rate of Return (IRR)
- Payback Period
- Data Visualization
- Interactive Charts
