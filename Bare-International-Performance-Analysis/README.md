# Bare International Performance Analysis Dashboard

## Project Overview

This project analyzes store performance evaluation data using Microsoft Excel. The dashboard provides insights into employee performance, regional trends, evaluation criteria, and overall service quality.

---

## Objectives

- Analyze performance scores
- Identify high and low performing regions
- Track performance trends over time
- Compare evaluation criteria
- Measure High Performer percentage

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Heatmap
- KPI Cards
- Excel Dashboard

## Key Insights

- Orissa and Punjab recorded the highest High Performer percentage.
- Closure & Care showed the lowest average score among evaluation criteria.
- Performance remained relatively stable throughout the evaluation period.
- Some states require targeted improvement initiatives.
